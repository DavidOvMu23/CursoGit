# Lenguajes

## Breve resumen del módulo
Este módulo abarca el estudio de diferentes lenguajes marcas, sus paradigmas y aplicaciones prácticas.

## Qué crees que aprenderás en ese módulo
Espero aprender las características y usos de varios lenguajes de marcas, así como las mejores prácticas para escribir código eficiente y mantenible.

## Qué esperas de ese módulo
Espero adquirir una comprensión sólida de los lenguajes de marcas más utilizados y cómo aplicarlos en proyectos reales.
